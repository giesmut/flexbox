Layouten per Flexbox

Flexbox ist der Kurzname für das "CSS Flexible Box Layout Module", einem CSS3-Modul, das speziell zum Erstellen von CSS-Layouts entwickelt wurde und das traditionelle Techniken wie Float, absolute Positionierung oder Inline-Block ersetzen soll.

CSS-Layoutmethoden im Überblick

float: left | right und clear

position: absoulte

display: inline-block

display: table

NEU = display:flex

Flexbox

  • Für die flexible Gestaltung von HTML-Elementen
  • Ohne feste Größenangaben per width/height
  • Ohne float oder position
  • Erleichtert die responsive Gestaltung von Webseiten
  • Ideal für Komponenten wie Navigation oder Formulare
  • Auch für Seitenlayouts geeignet

Historie

  • 07/2009 - display: box
  • 03/2012 - display: flexbox (noch im IE10 verankert!)
  • 09/2012 - display: flex
